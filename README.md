textlint-plugin-latex2e depends on [the LaTeX parser](https://github.com/tamuratak/latex-utensils) since v1.0.0!

## Installation

```
$ npm install textlint-plugin-latex2e
```

And add to `.textlintrc`

```
   plugins: ["latex2e"]
```
